# balrachana

> Educational Rhymes for Nepali Children

- Built with [Nuxt.js](https://nuxtjs.org)
- Hosted on Firebase

## Build Setup

```bash
# install dependencies
$ yarn install

# serve with hot reload at localhost:3000
$ yarn run dev

# build for production and launch server
$ yarn run build
$ yarn start

# generate static project
$ yarn run generate
```
